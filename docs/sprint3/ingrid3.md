## latexdraw
### Package latexdraw
LaTeXDraw é um editor gráfico de desenho para LaTeX. O LaTeXDraw pode ser usado para 1) gerar código PSTricks; 2) criar diretamente imagens em PDF ou PS. O LaTeXDraw funciona em Linux, Windows e Mac OS X.
